## 實習心得

在這份科技金融實習中，我初期對股市一無所知，但透過每日爬蟲分析ptt股票版，
不斷學習股票名詞和分析方法。到最後，我已經能夠思考分析股價的方法，
思考如何選擇特定時段的文本資料進行分析，以及如何計算情緒指數與股價漲跌相關性。
實習期間主要成果是利用ChatGPT分析文本情緒與股價漲跌相關性，雖存在改進空間，
但我逐漸培養出解決問題的能力。實習讓我深入了解科技金融以及增進資料分析領域的能力，
雖未完成整個系統，但期待未來再參與類似項目，
這段經驗充實了我的跨領域知識，培養了分析思維，增強了解決問題的信心。
